== AI Recommendation API

[[ai-preference-category]]
=== 사용자 선호 카테고리 기반 AI 추천 기념품 조회

사용자가 선호하는 카테고리를 기반으로 AI가 기념품을 추천합니다.

* 인증 필요
* 온보딩 과정에서 선택한 카테고리를 기반으로 추천합니다.
* 선택한 카테고리가 5개이면 각 카테고리별로 2개씩 추천하며, 최대 10개까지 추천됩니다.
* 선택한 카테고리가 적으면 추천 기념품 수가 적어질 수 있습니다.

[[ai-recent-upload]]
=== 사용자 최근 업로드 기념품 기반 AI 추천 기념품 조회

사용자가 최근에 업로드한 기념품 정보를 기반으로 AI가 유사하거나 연관된 기념품을 추천합니다.

* 인증 필요
* 최근 업로드한 기념품의 나라, 카테고리, 기념품 명 등을 분석하여 추천합니다.
* 최근 업로드 이력이 없으면 **빈 리스트**가 반환되며, AI 호출은 수행되지 않습니다.
* 추천 결과는 최대 10개까지 제공됩니다.
