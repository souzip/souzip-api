[[search-locations]]
=== 위치 검색

위치(국가/도시) 검색 API입니다. 한글 이름으로 정확히 일치하는 결과를 반환합니다.

[[search-features]]
==== 검색 특징

* **정확한 일치**: 입력한 키워드와 정확히 일치하는 도시명 또는 국가명만 검색
* **한글만 지원**: 영문 검색은 지원하지 않음
* **우선순위 정렬**: 도시 일치 > 국가 일치 > priority 순

[[search-city]]
==== 도시명으로 검색

도시 이름을 정확히 입력하면 해당 도시를 반환합니다.

operation::search/locations-exact-city[snippets='http-request,query-parameters,http-response,response-fields']

[[search-country]]
==== 국가명으로 검색

국가명으로 검색하면 해당 국가의 도시들이 priority 순으로 반환됩니다.

operation::search/locations-country[snippets='http-request,query-parameters,http-response,response-fields']

[[search-pagination]]
==== 페이지네이션

최대 10개(기본값)씩 반환됩니다. pageNo와 pageSize로 제어할 수 있습니다.

operation::search/locations-pagination[snippets='http-request,query-parameters,http-response,response-fields']
