[[auth-refresh]]
=== 토큰 재발급

Access Token이 만료되었을 때 Refresh Token을 사용하여 새로운 토큰을 발급받는 API입니다.

* Refresh Token 유효기간이 10일 초과 남았을 경우: Access Token만 재발급
* Refresh Token 유효기간이 10일 이하 남았을 경우: Access Token과 Refresh Token 모두 재발급
