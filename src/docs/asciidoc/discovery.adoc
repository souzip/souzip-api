[[discovery-country-top10]]
=== 나라별 추천 기념품 TOP 10

특정 나라에서 등록된 추천 기념품을 조회하는 API입니다.

NOTE: 조회 결과는 최근 생성된 기념품 10개까지 제공됩니다.
      (1차 개발 기간에서는 찜, 조회수 등의 기능이 없어 생성일을 기준으로 추천합니다.)
