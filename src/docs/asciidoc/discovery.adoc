== Discovery API

[[discovery-country-top10]]
=== 나라별 추천 기념품 TOP 10 (상위 국가 기준)

기념품이 많이 등록된 상위 국가(최대 10개)를 조회하고,
각 국가별로 최근 생성된 추천 기념품을 최대 10개까지 제공합니다.

NOTE: 상위 국가는 "기념품 등록 수(누적)" 기준으로 선정됩니다.
      각 국가의 기념품 목록은 최근 생성된 기념품 최대 10개까지 제공됩니다.

[[discovery-category-top10]]
=== 카테고리별 기념품 추천

특정 카테고리에 속하는 추천 기념품을 조회하는 API입니다.

NOTE: 조회 결과는 최근 생성된 기념품 10개까지 제공됩니다.
      (1차 개발 기간에서는 찜, 조회수 등의 기능이 없어 생성일을 기준으로 추천합니다.)
