[[search-api]]
== Search API

[[search-locations]]
=== 위치 검색

위치(국가/도시) 검색 API입니다. Elasticsearch를 사용하여 한글/영문 이름으로 검색할 수 있습니다.

==== 엔드포인트
[source,http]
----
GET /api/search/locations
----

==== 요청 파라미터
[cols="1,1,1,3"]
|===
|파라미터 |타입 |필수 |설명

|keyword
|String
|O
|검색 키워드 (한글/영문)

|pageNo
|Integer
|X
|페이지 번호 (0부터 시작, 기본값: 0)

|pageSize
|Integer
|X
|페이지 크기 (기본값: 20, 최대: 100)
|===
