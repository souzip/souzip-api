[[souvenir-api]]
== Souvenir API

[[souvenir-get]]
=== 기념품 조회

기념품 상세 정보를 조회하는 API입니다.

* 인증 불필요 (Public API)
* 삭제된(soft delete) 기념품은 조회할 수 없습니다.
* 인증 불필요하지만, 로그인한 사용자가 조회하면 내 기념품 여부(isOwned) 함께 반환

==== HTTP 요청
include::{snippets}/souvenirs/get-souvenir/http-request.adoc[]

==== HTTP 응답
include::{snippets}/souvenirs/get-souvenir/http-response.adoc[]

==== 응답 필드
include::{snippets}/souvenirs/get-souvenir/response-fields.adoc[]

NOTE: 기념품에는 위치 정보(위도/경도)와 업로드된 파일 목록이 함께 반환됩니다.

[[souvenir-create]]
=== 기념품 생성

사용자가 새로운 기념품을 등록하는 API입니다.

* 인증 필요
* multipart/form-data 요청
* 기념품 정보(JSON) + 이미지 파일 업로드 가능
* 파일은 선택 사항입니다.

==== HTTP 요청
include::{snippets}/souvenirs/create-souvenir/http-request.adoc[]

==== 요청 헤더
[cols="1,1,1,3"]
|===
|헤더 |타입 |필수 |설명

|Authorization
|String
|O
|Bearer {accessToken}
|===

==== Part (multipart/form-data에서 전달되는 파트)
include::{snippets}/souvenirs/create-souvenir/request-parts.adoc[]

NOTE: multipart/form-data 요청에서 JSON 형태의 기념품 정보는 souvenir 파트로 전달되며, 해당 필드는 아래의 ‘요청 필드’에서 문서화됩니다.
이미지 파일과 같은 바이너리 파일은 files 파트로 전달되며, 구조화된 필드가 없기 때문에 ‘요청 필드’에는 포함되지 않고, 바로 위의 Part 표에서 이름과 설명만 문서화됩니다.

==== 요청 필드 (souvenir JSON 안의 상세 필드)
include::{snippets}/souvenirs/create-souvenir/request-part-souvenir-fields.adoc[]

==== HTTP 응답
include::{snippets}/souvenirs/create-souvenir/http-response.adoc[]

==== 응답 필드
include::{snippets}/souvenirs/create-souvenir/response-fields.adoc[]

NOTE: 가격은 현지 통화 또는 원화 기준으로 입력할 수 있으며, 서버에서 환율이 자동 적용됩니다.

[[souvenir-nearby]]
=== 근처 기념품 조회

사용자의 위도/경도 기준으로 특정 반경 내 기념품을 조회하는 API입니다.

* 인증 불필요 (Public API)
* 삭제된(soft delete) 기념품은 조회할 수 없습니다.
* 거리 단위는 미터(m)이며, 서버에서 계산된 거리를 함께 반환합니다.

==== HTTP 요청
include::{snippets}/souvenirs/get-nearby-souvenirs/http-request.adoc[]

==== 요청 헤더
[cols="1,1,1,3"]
|===
|헤더 |타입 |필수 |설명

|Authorization
|String
|O
|Bearer {accessToken}
|===

==== 요청 파라미터
[cols="1,1,1,3"]
|===
|파라미터 |타입 |필수 |설명

|latitude
|Decimal
|O
|위도 (decimal, 소수점 7자리까지)

|longitude
|Decimal
|O
|경도 (decimal, 소수점 7자리까지)

|radiusMeter
|Integer
|X
|조회 반경 (미터 단위, 기본값 4000m)
|===

include::{snippets}/souvenirs/get-nearby-souvenirs/http-response.adoc[]

==== 응답 필드
