[[user-onboarding]]
=== 온보딩 완료

신규 사용자가 온보딩을 완료하고 약관 동의, 닉네임, 프로필 이미지 색상, 관심 카테고리를 설정하는 API입니다.

* 온보딩은 최초 1회만 가능합니다.
* 필수 약관에 모두 동의해야 합니다 (만 14세 이상, 서비스 이용약관, 개인정보 수집 및 이용, 위치기반 서비스)
* 마케팅 수신 동의는 선택 사항입니다.
* 닉네임은 2~11자까지 입력 가능하며, 한글/영문/숫자만 사용 가능합니다.
* 프로필 이미지는 색상(red, blue, yellow, purple) 중 선택합니다.
* 카테고리는 최소 1개, 최대 5개 선택이 가능합니다.
