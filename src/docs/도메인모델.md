# Souzip 도메인 모델

## 도메인 모델 만들기
- 듣고 배우기
- '중요한 것'들 찾기 (개념 식별)
- '연결 고리' 찾기 (관계 정의)
- '것'들을 설명하기 (속성 및 기본 행위 명시)
- 그려보기 (시각화) <br/>
  예시) 클래스 다이어그램
- 이야기 하고 다듬기 (반복)

## 도메인 모델

### 위치 (Location)
_Entity_

#### 속성
- `id`: `Long`
- `name`: 장소명
- `address`: 주소
- `latitude`: 위도
- `longitude`: 경도
- `createdAt`: 등록일
- `updatedAt`: 수정일

#### 행위
- `static create()`: 위치 생성

#### 규칙
- 좌표는 유효한 범위여야 한다. (위도: -90~90, 경도: -180~180)
