# Souzip 도메인 모델

## 도메인 개요
Souzip은 여행지 정보를 제공하는 서비스입니다.
- 사용자는 **도시, 국가, 장소**를 검색할 수 있습니다.
- **데이터베이스**에 저장된 도시 정보를 우선 검색합니다.
- DB에 없는 경우 **Google Places API**를 통해 실시간 검색합니다.

---

## 도메인 모델

### 위치 (Location)
_Entity_

#### 속성
- `id`: `Long`
- `name`: 장소명
- `address`: 주소
- `latitude`: 위도
- `longitude`: 경도
- `createdAt`: 등록일
- `updatedAt`: 수정일

#### 행위
- static create(name, nameEn, address, latitude, longitude): 위치 생성

#### 규칙
- 좌표는 유효한 범위여야 한다. (위도: -90~90, 경도: -180~180)
