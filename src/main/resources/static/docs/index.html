공통 응답 구조

성공 응답

{
  "data": { ... },
  "message": "성공 메시지"
}

에러 응답

테스트 환경에선 `traceId`가 포함되지 않아 API 문서에는 포함되지 않습니다.
일반 비즈니스 에러
{
  "traceId": "a1b2c3d4",
  "message": "만료된 Refresh Token입니다."
}
Validation 에러
{
  "traceId": "a1b2c3d4",
  "message": "에러 메시지",
  "errors": [
    {
      "field": "필드명",
      "value": "입력값",
      "reason": "에러 사유"
    }
  ]
}

Authentication API

소셜 로그인

OAuth를 통한 로그인 API입니다. 신규 사용자의 경우 자동으로 회원가입이 진행됩니다.

엔드포인트

POST /api/auth/login/{provider}

경로 파라미터

provider

String

O

OAuth Provider (kakao, google, apple)

토큰 재발급

Access Token이 만료되었을 때 Refresh Token을 사용하여 새로운 토큰을 발급받는 API입니다.

  • Refresh Token 유효기간이 10일 초과 남았을 경우: Access Token만 재발급

  • Refresh Token 유효기간이 10일 이하 남았을 경우: Access Token과 Refresh Token 모두 재발급

Content-Type: application/json
